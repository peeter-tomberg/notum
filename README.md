# Notum
--------------


## Installation
```
bower install notum
```

### Setup requirejs
```javascript
    var requirejs = {
        "paths": {
            "Notum" : "bower_components/notum/src"
        }
    };

```

### Setup
```javascript
    var Notum = require('Notum/Notum');
    var notum = new Notum();
```
